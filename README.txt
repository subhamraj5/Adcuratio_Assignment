Following are the description of the files:

1.Users.py     : This file contains the user bean and all its related methods

2.Resources.py : This file contains the resource class which maintains the different resources

3.Roles.py     : This files contains role class which contains different roles related to user

4.ActionType.py   : This files contains the action bean of three cations namely read,write and delete

5.Main.py      : This is the driver file in which creation of all users,roles,resources and action type takes place and represent role based user access of resources
